# Triple Espresso

Esta é a página da Triple Espresso, desenvolvida com HTML e CSS.
A estrutura inclui uma seção de receitas, na qual utilizamos a tag iframe para incorporar vídeos externos diretamente na página.
Além disso, há uma seção de reservas implementada com a tag form, permitindo a coleta e o envio de dados do usuário.

## Recursos do projeto

- HTML5 semântico
- Flexbox
- Posicionamento
- Incorporação de conteúdo
- Formulário
- Uso de BEM
